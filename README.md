# Road Traffic Analysis

Analyse des accidents de la route - Dataset RTA Ethiopia (Kaggle).
12 316 accidents analysés.

## Objectif
Comprendre les causes et conditions des accidents de la route
pour améliorer la sécurité routière.

## Outils utilisés
- Python (pandas, matplotlib)
- SQL (SQLite)
- Power BI
- Excel

## Fichiers
- rta_python.ipynb : analyse exploratoire en Python
- rta_sql.ipynb : analyse SQL avec SQLite
- rta_dashboard.pbix : dashboard Power BI
- rta_analyse.xlsx : données nettoyées Excel

## Résultats clés
- 158 accidents mortels (1.28%)
- Vendredi jour le plus dangereux
- Principale cause : manque de distance
- 91% des accidents sur routes asphaltées
